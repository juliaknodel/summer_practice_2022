Программа для обработки данных бисульфитного секвенирования.

Выполняемые шаги:
- fastp
- bismark
- deduplicate_bismark
- bismark_methylathion_extractor

Описание структуры:
1. Все параметры задаются в файле wrapper_conf.conf
2. Оберточный скрипт wrapper.sh - выполняет скрипты из папки scripts_for_wrapper/
3. wrapper.sh может принимать на вход 3 аргумента (необязательных):
	1 - Путь до выходной директории (будет заменен параметр OUTPUT_DIR из конфигурационного файла)
	2 - Путь до файла с ридами _1 (будет заменен параметр SAMPLE_1_PATH из конфигурационного файла)
	3 - Путь до файла с ридами _2 (будет заменен параметр SAMPLE_2_PATH из конфигурационного файла)
4. Скрипт wrapper_sbatch.sbatch - для запуска wrapper.sh через sbatch 


ПРИМЕР ЗАПУСКА БЕЗ SBATCH

	- С аргументами:
	  $ ./wrapper.sh path_to_output_dir sample_1.fq.gz sample_2.fq.gz

	- Аргументы могут быть опущены (будут взяты значения из конфигурационного файла wrapper_conf.conf):
	  $ ./wrapper.sh



ПРИМЕР ЗАПУСКА С SBATCH

	- С аргументами
	  $ ./wrapper_sbatch.sbatch job_name path_to_output_dir sample_1.fq.gz sample_2.fq.gz

	- Без аргументов (однако первый аргумент job-name обязателен)
	  $ ./wrapper_sbatch.sbatch job_name

Для запуска нескольких job через sbatch можно просто делать вызовы (как в примере) из цикла, но обязательно давать различные job_name


Результаты запуска будут лежать в заданной выходной директории (OUTPUT_DIR), для каждого шага - своя папка.
